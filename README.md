# Apurv Sharma - Portfolio Website

A modern, animated portfolio website built for GitHub Pages.

## 🚀 Quick Setup

1. **Create a GitHub repository** named `Apurvsharma1.github.io` (your username + .github.io)
2. **Upload these files** to the repository:
   - `index.html` (main website file)
   - `profile.png` (your profile photo)
3. **Enable GitHub Pages**:
   - Go to Settings → Pages
   - Select "main" branch
   - Click Save
4. **Visit your site**: `https://Apurvsharma1.github.io`

## ✏️ Things to Customize (Optional)

Open `index.html` in any text editor and update these:

### 1. Google Scholar Link (Line ~475)
Replace the `#` with your actual Google Scholar profile URL:
```html
<a href="YOUR_GOOGLE_SCHOLAR_URL_HERE" class="contact-link" target="_blank">
```

### 2. Profile Photo
- The file is named `profile.png` - your photo is already included!
- If you want to use a different photo, just replace `profile.png` with your new image
- Make sure to keep the same filename OR update line ~235 in index.html:
```html
<img src="profile.png" alt="Apurv Sharma" class="profile-img">
```

### 3. Add More Content (Optional)
You can easily add more:
- Projects (around line 310-360)
- Publications (around line 445-470)
- Achievements (around line 500-550)

Just copy the existing card structure and modify the content.
